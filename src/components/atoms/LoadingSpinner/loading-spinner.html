How to Use

Copy the SVG spinner code and adjust the container size using Tailwind classes (w-*, h-*): 

<!-- Example: Medium Spinner (32px) -->
<div class="spinner w-8 h-8">
  <svg class="w-full h-full" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
    <path class="spinner-track"
      d="M 50 5 L 75 5 A 20 20 0 0 1 95 25 L 95 75 A 20 20 0 0 1 75 95 L 25 95 A 20 20 0 0 1 5 75 L 5 25 A 20 20 0 0 1 25 5 L 50 5 Z" />
    <path class="spinner-anim"
      d="M 50 5 L 75 5 A 20 20 0 0 1 95 25 L 95 75 A 20 20 0 0 1 75 95 L 25 95 A 20 20 0 0 1 5 75 L 5 25 A 20 20 0 0 1 25 5 L 50 5 Z" />
  </svg>
</div>

Design:

  • Shape: Rounded square (20px corner radius)
  • Track color: grey-tertiary (#EDEDED)
  • Active stroke: green-primary (#03683D)
  • Animation: 1.5s linear stroke dash animation
